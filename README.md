# PetCare dashboard

El codigo está deployeado en el siguiente link: https://edmachina.ronbachmann.dev

## Requisitos
- Node.js: Se recomienda usar la versión 20.19.0 o una más nueva.

## Instalacion
1. Clonar el repositorio.
2. Abrir una terminal en la carpeta del proyecto.
3. Instalar dependencias.
```bash
npm install
```

## Como correr el codigo?
1. Ejecutar
```bash
npm run dev
```
2. Abrir `http://localhost:5173` en el navegador

## Aclaraciones
- Como no estaba especificado en los requisitos, por cuestiones de tiempo no me concentré demasiado en hacer que la interfaz
  fuera responsive.
- La interfaz fue implementada pensando exclusivamente en una pantalla con
  resolución de 1920x1080.
